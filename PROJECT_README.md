## KEY PROMPT COMPONENTS (Remember These)

Every industry-grade prompt needs:

1. **ROLE** - Define the AI's expertise and background
2. **TASK** - State exactly what the AI should do
3. **CONTEXT** - Provide relevant background information
4. **OUTPUT** - Specify format, length, style, tone
5. **CONSTRAINTS** - Clearly state what to do and avoid
6. **EXAMPLES** - Show sample input/output (when helpful)

**Pro Tip:** These 6 components are the backbone of every successful prompt.

## COMMON MISTAKES TO AVOID

| Mistake | Solution |
|-----------|-----------|
| Vague instructions | Use specific, measurable language |
| Missing context | Provide sufficient background details |
| No format spec | Explicitly define output structure |
| Too many goals | Focus on one primary task |
| Contradictions | Review and resolve conflicts |
| No examples | Include 1-2 reference examples |
| Ignoring constraints | Explicitly list do's and don'ts |
